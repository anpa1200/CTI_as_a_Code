# AI-Assisted Analysis Outputs

This directory stores outputs from AI-assisted analysis tools used during the investigation.

**What goes here:**
- LLM-assisted timeline synthesis outputs (reviewed and corrected by analyst)
- AI-generated Sigma rule drafts (before analyst review)
- Automated ATT&CK mapping suggestions (validated against evidence before use)

**Usage note:** All AI outputs must be reviewed and validated by a human analyst before inclusion in deliverables. Never cite AI outputs as evidence — they are analytical aids, not evidence sources.
