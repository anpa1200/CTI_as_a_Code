# Processed Evidence

This directory holds normalized/processed versions of raw evidence artifacts from the March 2025 NDSA eID breach.

**What goes here:**
- Timesketch timeline exports covering the 16 A05 timeline events
- Hayabusa processed `.evtx` summaries (VRID-SRV-01, JUMPHOST-CONTRACTOR-01)
- Velociraptor VQL query result exports from post-incident collection
- Database audit log normalized exports

**Source material:** Raw artifacts are in `../raw/`

**Classification note:** All processed artifacts should carry CONFIDENTIAL header; GOVNET classified segment evidence is NOT stored in this repository.
