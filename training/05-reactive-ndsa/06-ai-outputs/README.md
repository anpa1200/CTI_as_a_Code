# AI-Assisted Analysis Outputs

This directory stores outputs from AI-assisted analysis tools used during the NDSA eID breach investigation.

**What goes here:**
- LLM-assisted timeline correlation suggestions (validated against A05 timeline events before use)
- AI-generated Sigma rule drafts (reviewed by SOC Engineering before deployment)
- Automated ATT&CK mapping suggestions for Iranian-nexus TTP correlation

**Classification note:** AI tools used for this analysis must not process GOVNET classified content. Only UNCLASSIFIED/CONFIDENTIAL artifacts should be processed through external AI services.
