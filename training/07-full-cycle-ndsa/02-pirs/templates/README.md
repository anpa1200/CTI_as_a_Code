# PIR Templates

This directory holds blank PIR (Priority Intelligence Requirement) templates for the NDSA CTI program.

**Classification:** Templates are UNCLASSIFIED; populated PIRs may be CONFIDENTIAL.

**Template to use when adding a new PIR:**
```markdown
## PIR-XXX: [Short Title]

**Owner:** [NDSA stakeholder — DG / CISO / SOC Lead / Legal]
**INCD reference:** [INCD-CID Section number if compliance-driven]
**Review cycle:** [Weekly / Monthly / Triggered]
**Status:** Active | Suspended | Satisfied

### Requirement Statement
[One sentence: "We need to know X in order to Y."]

### Key Intelligence Questions (KIQs)
1. [Specific actionable question]
2. [Specific actionable question]

### Collection Priorities
- Source: CERT-IL MISP | Priority: High / Medium / Low
- Source: INCD feed | Priority: —

### Satisfaction Criteria
[What would constitute a fully answered PIR?]
```

**Existing PIRs:** See `../pir-register.md` for all active PIRs.
