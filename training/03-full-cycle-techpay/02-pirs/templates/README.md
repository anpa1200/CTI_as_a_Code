# PIR Templates

This directory holds blank PIR (Priority Intelligence Requirement) templates for the TechPay CTI program.

**Template to use when adding a new PIR:**
```markdown
## PIR-XXX: [Short Title]

**Owner:** [Stakeholder name and role]
**Review cycle:** [Weekly / Monthly / Triggered]
**Status:** Active | Suspended | Satisfied

### Requirement Statement
[One-sentence intelligence requirement: "We need to know X in order to Y."]

### Key Intelligence Questions (KIQs)
1. [Specific question 1]
2. [Specific question 2]

### Collection Priorities
- Source: [Source name] | Priority: [High / Medium / Low]

### Standing Intelligence Requirements (SIRs)
- SIR-XXX: [Specific recurring collection task]

### Satisfaction Criteria
[What would constitute a fully answered PIR?]
```

**Existing PIRs:** See `../pir-register.md` for all active PIRs.
