# MOU Templates

This directory holds Memorandum of Understanding templates for intelligence sharing partnerships.

**Status:** CERT-IL MOU is the primary pending partnership. See `../sharing-architecture.md` for draft terms.

**Template structure (TLP:AMBER — standard bilateral MOU):**
```
INTELLIGENCE SHARING MEMORANDUM OF UNDERSTANDING

Parties: [Organization A] and [Organization B]
Classification: CONFIDENTIAL
Effective date: [date]
Review date: [date + 12 months]

1. PURPOSE
   [1-2 sentences on what intelligence will be shared and why]

2. SCOPE
   - Information types: [IOCs / TTPs / Sector threat reports / Incident data]
   - TLP levels permitted: [WHITE / GREEN / AMBER — RED requires separate agreement]
   - Exclusions: [What is NOT shared — classified material, PII, etc.]

3. HANDLING REQUIREMENTS
   - Minimum: TLP:AMBER — share within named organizations only
   - Storage: Encrypted at rest; access limited to CTI team
   - Retention: [X months]

4. INCIDENT NOTIFICATION
   - [Organization A] will notify [Organization B] within [X hours] of confirmed incident
   - INCD pre-notification takes precedence for INCD-regulated entities

5. REVIEW
   Annual review; either party may terminate with [30 days] written notice

Signed: _____________ Date: _________
```
