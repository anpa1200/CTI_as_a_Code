# AI-Assisted Outputs

All files in this folder were produced with AI assistance. They require analyst review before use.

## Policy

- AI outputs are **drafts only** — never publish directly without review.
- Label every AI-assisted document with `[AI-ASSISTED DRAFT — REVIEW REQUIRED]` in the header.
- Record which model was used and the date generated.
- Document what was changed during review.

## Appropriate Uses of AI in CTI Work

- First-pass timeline reconstruction from raw logs
- Draft detection rule logic from TTP descriptions
- Initial executive brief structure and language
- IOC enrichment lookups and summarization
- Translating technical findings into non-technical language

## Inappropriate Uses

- Attribution claims (AI cannot assess evidence reliability)
- Classification decisions
- Regulatory notification content (legal review required)
- Confidence level assignment (requires human analyst judgment)
